I guess it's time to start deploying the household agents. Shannon and I decided a while ago to call them "bears" after a discussion about the [man or bear meme](https://www.seattletimes.com/life/why-women-prefer-the-bear-in-manvsbear/).

I spent too much time thinking about this over the last few months. I've suspected for a long time that we needed to focus on the agent layer rather than fine-tuning an open-weights model, but only when I read [the MemGPT paper](https://arxiv.org/abs/2310.08560) did I really get that memory could be properly decoupled from model knowledge enough that we could build up something of long-term value.

That led me to [the MemGPT team's explainer](https://www.deeplearning.ai/short-courses/llms-as-operating-systems-agent-memory/) and I decided to deploy [Letta](https://www.letta.com/).

This is also an opportunity to [vibe](https://xcancel.com/karpathy/status/1886192184808149383) after learning to trust Kilo Code with some scoped admin tools for Hexmap.

So I've started with some infrastructure-as-code. (Or do we call it "gitops" now?) LiteLLM and Letta are up and running.
